# MSI Afterburner Auto-Setup Script

Этот скрипт на PowerShell автоматически скачивает и устанавливает актуальную версию MSI Afterburner (на данный момент 4.6.7 Beta 2) напрямую с официального сервера MSI, а также настраивает конфигурационные файлы и профили.

## 🚀 Особенности

- Скачивание архива `MSIAfterburnerSetup.zip` по **прямой и надежной ссылке** с серверов MSI.
- Автоматическая распаковка архива средствами Windows PowerShell (`Expand-Archive`).
- Запуск установщика в **тихом (silent) режиме**, без необходимости нажимать "Далее".
- Автоматическое создание папки `Profiles` в директории установки программы.
- Преднастроенная подробная конфигурация программы (графики, управление вентиляторами, отключение ненужных лимитов и многое другое) — вшита прямо в код скрипта.

## ⚙️ Требования

- Windows 10 или Windows 11.
- Права Администратора. Скрипт **автоматически** запросит повышение прав (`UAC`) при запуске.

## 🛠 Использование

1. Скачайте файл скрипта `3 MSI Afterburner.ps1` на свой компьютер.
2. Кликните по файлу правой кнопкой мыши и выберите **«Выполнить с помощью PowerShell»** (Run with PowerShell).
3. Дождитесь завершения работы скрипта:
   - Он скачает MSI Afterburner в папку `%TEMP%`.
   - Распакует и установит программу.
   - Запишет все необходимые настройки в конфигурационные файлы.

*Обратите внимание: скрипт выдаст красное предупреждение в консоли, уведомляя об отключении мониторинга 'Power' и 'Power Percent' (это сделано специально для предотвращения некоторых проблем с FPS и просадками 1% low).*

## ⚠️ Отказ от ответственности (Disclaimer)

Точная настройка кривой скорости вентиляторов и параметров мониторинга зашита в сам скрипт. Вы можете открыть файл `3 MSI Afterburner.ps1` в любом текстовом редакторе и отредактировать блок настроек `[Settings]` так, как вам нужно. Используйте встроенную конфигурацию на свой страх и риск.
